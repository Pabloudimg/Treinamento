O que é HTML?

HTML é uma linguagem de marcação utilizada na construção de páginas na Web.

Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. HyTime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo.

Clique aqui para maiores informações!


Tags Semânticas

O html oferece outras tags container para faciliar a interpretação de documentos html. Tags semânticas dao significado ao conteúdo. Elementos "conteiners"

Exemplos de tags: HEADER, SECTION, NAV, MAIN, FOOTER, DETAILS, SUMMARY, ADDRESS, ASIDE, DIV...
